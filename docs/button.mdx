## Mode

the `mode` prop determine the color of the button, the default value of `mode` is `normal`.

<Playground>
  <Button mode="primary">Enter</Button>
  <Button mode="danger">Leave</Button>
  <Button mode="normal">Cancel</Button>
</Playground>

## Loading

<Playground>
  {()=>{
    const [loading,setLoading]=useState(false);
    return (
      <Button 
        loading={loading}
        onClick={()=>setLoading(!loading)}
      >
        Click to Load
      </Button>
    )
  }}
</Playground>
