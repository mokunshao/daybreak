## Basic

`CodePreview` has no code highlighting, just like `Stack Overflow`.

`CodePreview` 没有代码高亮，就像 `Stack Overflow` 一样。

<Playground>
  {()=>{
    const code = `
public class Main {
    public static void main(String[] args) {
    System.out.println("Hello, World!");
  }
}
    `;
    return (
      <div>
        <CodePreview code={code.trim()} />
      </div>
    );
  }}
</Playground>
