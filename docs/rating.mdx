
## Basic usage

<Playground>

## Clearable

When you click on a star repeatedly, the value will be set to 0.

当你反复点击一颗星星时，值将会被设置为 0。

<Playground>
{() => {
  const [value, setValue] = useState(3);
  return (
    <div>
      <Rating clearable value={value} max={6} onChange={(val) => setValue(val)} />
    </div>
  );
}}
</Playground>

## Props

| Prop | type | 
| --- | --- | 
| value | number |
| max | number |
| onChange | (value: number) => void |
| clearable | boolean |
